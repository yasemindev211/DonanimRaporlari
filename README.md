Bilgisayar Mimarisi (CPU, RAM, Cache)
⚙️ CPU Nedir?

CPU (işlemci), bilgisayarın beynidir.

 Temel Bileşenler
ALU (Arithmetic Logic Unit): Matematiksel ve mantıksal işlemleri yapar
Control Unit: Komutları okur ve sistemi yönetir
 Cache (Önbellek)

CPU ile RAM arasındaki hızlı hafızadır.

L1: En hızlı, en küçük
L2: Orta seviye
L3: Daha büyük, çekirdekler arası paylaşılır

 Sık kullanılan veriler burada tutulur → performans artar

 RAM Nedir?

Geçici bellektir. Programlar çalışırken veriler burada tutulur.

DRAM: Daha yaygın, yenilenmesi gerekir
SRAM: Daha hızlı, cache içinde kullanılır
 Neden Önemli?
CPU ↔ RAM gecikmesini anlamayı sağlar
Performans optimizasyonu yapmayı kolaylaştırır
Kodun donanımda nasıl çalıştığını kavratır
Özet

Hız sırası: Cache > RAM > Disk
